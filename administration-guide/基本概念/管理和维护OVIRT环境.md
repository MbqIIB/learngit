# 管理和维护OVIRT环境

OVIRT 管理系统需要一个系统管理员来保持整个系统的良好运行。作为一个
系统管理员，他需要负责以下的一些工作:

-   管理物理资源和虚拟资源。包括升级主机，添加新的主机，导入存储域,
    管理虚拟机池等等

-   监控整个系统的运行情况，发生一些严重问题时候一边可以很快恢复。
    比如某些主机负载太高，内存或者存储空间不够等等。

-   满足普通用户对虚拟机使用的一些需求，比如升级虚拟机的操作系统，给虚拟机分配更多的内存等。

-   使用 tags 管理一些特性的对象属性

-   管理搜索结果，把它们作为公共的书签

-   管理用户的创建和权限

-   为一些用户或者虚拟机错误操作恢复环境

-   生成报告

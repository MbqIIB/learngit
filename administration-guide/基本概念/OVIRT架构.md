# OVIRT架构

OVIRT系统是由以下组件所组成的:

-   用来运行虚拟机的主机 利用了 KVM 来提供虚拟化的功能。

-   代理程序和工具 包括 VDSM，QEMU 和
    libvirt。这些工具提供了对本地虚拟机，网络，存储等的管理。

-   OVIRT虚拟化管理工具 是 OVIRT
    系统的管理平台。它提供了一个图形化的管理界面，方便用户来管理 OVIRT
    系统

-   存储域 用来存储虚拟机的镜像资源，模板，和 ISO 文件等。

-   数据库 用来保存系统的信息。

-   外部目录服务器 用来提供用户权限管理

-   网络管理 管理整个环境的网络，包括物理网卡的管理，和逻辑网络的管理

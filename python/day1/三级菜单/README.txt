# 1> 根据菜单提示输入要查询得城市序号进行查询
# 2> 在任何一级菜单中输入"b":返回上一级菜单
# 3> 在任何一级菜单中输入"q":退出
# 4> 输入不是"q" or "b" 将不会进入下一步查询,提示用户输入有误,并重新打印菜单提示用户重新输入
# 5> 输入得数字不在提示范围内 将不会进入下一步查询,提示用户输入有误,并重新打印菜单提示用户重新输入
# 6> 整个系统可以反复查询,不输入"q"将不会退出程序
